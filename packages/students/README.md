# Packages

This is the final state of the project. In this version, we organized our source files into packages. Although all the `.java` files are stored in the same directory, compiling them will create a `cop3330` folder with several subdirectories. These subdirectories correspond to the package declarations in each class and will contain the generated `.class` files accordingly.
